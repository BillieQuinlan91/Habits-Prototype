# Becoming User Testing

Use the hosted demo build for round-one product testing. Keep auth out of the session so feedback stays focused on the product loop.

## Recommended Participants

- 3 founders or operators
- 2 community builders or founder-community hosts

Keep each session to 20 minutes.

## Session Tasks

Ask each participant to:

1. Join or create a tribe
2. Pick habits they would actually track
3. Log today&apos;s habits
4. Check tribe performance
5. Encourage one tribe member
6. Adjust a habit or settings from Profile

## What to Measure

- Time to first useful action
- Where they hesitate, backtrack, or ask for explanation
- Whether they understand the Sunday ritual without prompting
- Whether they would return daily and weekly

## Feedback Buckets

- Confusion
- Motivation
- Trust / completeness

## Testing Notes

- Round 1: demo mode only, focus on comprehension and emotional pull
- Round 2: persistence plus Sunday ritual fidelity
- Round 3: real auth and live tribe data with a small community
